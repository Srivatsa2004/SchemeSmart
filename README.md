# SchemeSmart
**Overview:**
A full-stack application that recommends government schemes based on user details using Machine Learning.

## Tech Stack
- Frontend: React
- Backend: Spring Boot (Java)
- Database: MongoDB
- ML Model: Random Forest (Python, Flask)

## Architecture
React → Spring Boot → Flask API → ML Model → MongoDB

## Features
- User input-based recommendations
- ML-powered prediction
- Real-time API integration
- Microservice architecture

## ML Approach
- Random Forest classifier
- Trained on synthetic dataset
- Features: age, income, occupation, category, state

##  Limitations
- Small dataset
- Synthetic training data


##  Future Improvements
- Real government dataset
- Multiple scheme recommendations
- Ranking system
- Improved UI
